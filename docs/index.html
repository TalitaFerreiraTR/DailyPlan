Instalar Extensão Chrome

A extensão permite ler dados automaticamente das abas da SS, PSAI e SAI no navegador Chrome. O site funciona sem ela, mas a extensão agiliza o trabalho.

Passo a passo
  1. Clique no botão abaixo para baixar os arquivos da extensão
  2. Extraia o arquivo .zip em uma pasta no seu computador
  3. No Chrome, acesse chrome://extensions
  4. Ative o Modo do desenvolvedor (canto superior direito)
  5. Clique em Carregar sem compactação
  6. Selecione a pasta onde extraiu os arquivos
Baixar Extensão (.zip)
⚠ Após extrair o zip, selecione a pasta DailyPlan-main → extensao no passo 6. Não selecione a pasta raiz.

Clique em um usuario para visualizar seus dados (somente leitura).
